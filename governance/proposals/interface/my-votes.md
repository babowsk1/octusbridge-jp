# 自分の投票

［**自分の投票**］は、あなたの投票可能性に関する情報を表示しています：

* 投票力 - 投票数 (ステークされたBRIDGEトークンに比例して)
* 投票の重み - 投票結果に対する潜在的な影響力 (全ステークホルダーの総投票数に対するあなたの割合)
* 投票された提案 - 投票した提案数
